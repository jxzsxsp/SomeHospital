# 某类型医院
某类型医院致力于帮助您查询您身边的“某类型医院”信息。

* 数据来源于Github上的[Hospital](https://github.com/open-power-workgroup/Hospital)提供的开放数据。
* 地址信息来自于高德Geo反查。
* 软件组织架构使用MVP，参考Google [todo-mvp](https://github.com/googlesamples/android-architecture)

如有不足还望不吝指点！希望共同进步！
